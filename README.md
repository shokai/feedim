feedim
======

feeds to IM. (using im.kayac.com)


Dependencies
============

* TokyoCabinet
* Twitter


Setup
=====

git clone

    % git clone git://github.com/shokai/feedim.git


Install Dependencies
--------------------

    # Mac OS X
    % sudo port install tokyocabinet
  
    # Install gems
    % bundle install


Config
------

    % cp sample.config.yaml config.yaml

then edit it.


Run
===

    % ruby feedim.rb
